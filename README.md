Projeto e Desenvolvimento de um Manipulador Robótico Antropomórfico de 6 GDL (Parcialmente Atuado) para o Ensino da Cinemática Direta

Descrição

Este repositório reúne os arquivos de apoio ao projeto de um manipulador robótico antropomórfico de seis graus de liberdade (6 GDL), parcialmente atuado e impresso em 3D, desenvolvido com finalidade didática para o ensino de cinemática direta.
O projeto foi concebido com foco em:
- visualização prática de conceitos de cinemática direta;
- integração entre modelagem mecânica, eletrônica embarcada e interface homem-máquina;
- validação experimental por comparação entre resultados obtidos via firmware embarcado, MATLAB e visão computacional com marcadores ArUco. 
Observação: embora o manipulador possua 6 GDL, o protótipo é parcialmente atuado, com 3 juntas motorizadas e instrumentadas para os experimentos didáticos, enquanto as demais juntas permanecem fixas no escopo proposto. 

Conteúdo deste repositório

Este repositório foi organizado para concentrar os materiais técnicos utilizados no projeto, incluindo:
- Código-fonte do sistema embarcado (Firmware)
- Controle de motores, leitura de sensores, comunicação com interface e cálculo de cinemática direta embarcada.
- Código-fonte de validação com ArUco
- Rotinas de visão computacional para medição/estimativa de pose e comparação com os resultados do sistema.

Desenhos de referência em SolidWorks
- Desenhos de peças (elos, base, engrenagens etc.)
- Desenho de montagem de conjuntos
- Referências dimensionais para fabricação e montagem do protótipo

Documentação complementar (opcional)
- Esquemático EasyEDA
- Interface IHM Nextion
- Desenho de apoio 2D
  
Materiais auxiliares, imagens, PDFs e instruções para reprodução parcial do sistema.
